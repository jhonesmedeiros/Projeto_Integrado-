🎉 Abracadabra Kids
Sistema de Gestão para Recreação Infantil

O Abracadabra Kids é um aplicativo desenvolvido para auxiliar empresas de recreação infantil no gerenciamento de clientes, eventos, agenda e controle financeiro, utilizando conceitos de Computação em Nuvem e Qualidade de Software.

🚀 Tecnologias Utilizadas
Flutter
Dart
Firebase Authentication
Cloud Firestore
GitHub
📱 Funcionalidades
Login de usuários
Cadastro de clientes
Agendamento de eventos
Controle financeiro
Agenda de atividades
Relatórios
☁️ Computação em Nuvem

O sistema utiliza o Firebase para armazenar dados na nuvem, garantindo:

Acesso remoto
Segurança
Backup automático
Escalabilidade
🔒 Qualidade de Software

O projeto segue princípios da ISO 25010, priorizando:

Funcionalidade
Usabilidade
Segurança
Confiabilidade
Manutenibilidade

Desenvolver uma solução tecnológica para otimizar a gestão da empresa Abracadabra Kids, centralizando informações e melhorando a organização dos serviços prestados.
